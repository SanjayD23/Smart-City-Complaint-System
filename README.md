# Smart City Complaint Management System

A full-stack web application for managing municipal complaints with role-based access control, real-time tracking, image uploads, and automated email notifications.


Key Features

Citizens – Submit complaints with photos & location, track status

Admins – Assign complaints to departments, monitor progress

Officers – Update status, resolve issues

Google Maps Integration – Location-based complaint submission

Automated Email Notifications – Updates at every stage


 Tech Stack

Backend: Node.js, Express, TypeScript, PostgreSQL, JWT, Bcrypt, Multer, Nodemailer
Frontend: HTML5, CSS3, JavaScript, Google Maps API
Deployment: Railway (Backend + DB), Vercel (Frontend)
